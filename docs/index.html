rclone, with a native Mac interface

Rclone Next is a menu bar app that wraps your existing rclone install. Browse remotes, mount drives in Finder, and run sync jobs without living in the terminal.

Download for macOS View on GitHub
macOS 14+ Free & open source Apache 2.0

Features

Menu bar app

Quick panel for mounts and transfers, plus a full manager window when you need it.

Browse remotes

Breadcrumb navigation, cached listings, upload, download, delete, rename, and public links.

Mount in Finder

Mount via macFUSE with saved mounts and auto-remount on launch.

Sync jobs

Copy, Sync, and Move with dry-run preview before you commit.

Your rclone config

Uses the rclone binary and remotes you already configured. Nothing to migrate.

Dashboard

Storage gauges, version info, and update checks for the app and rclone.

Get started

Requirements

rclone installed macFUSE for mounting macOS Sonoma (14) or later
  1. Download the latest RcloneNext.app.zip from GitHub Releases.
  2. Unzip and move Rclone Next to Applications.
  3. Open the app. Look for the icon in the menu bar (top right). Right-click → Open the first time if macOS blocks it.
  4. Install rclone and macFUSE if you have not already.
